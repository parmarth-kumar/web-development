# Mini Projects Notes

## Important Concepts & Tags
- HTML structural composition.
- Semantic layout elements.
- Nesting lists, tables, and multimedia objects.

## Common Mistakes
- Adding visual styling hacks (e.g. multiple `<br>` tags or font colors) instead of relying on CSS for spacing.
- Broken paths for local asset links.

## Interview Notes
- **How should we structure a website before writing CSS?** Always draft clean, semantic HTML layout skeletons first, ensuring proper document hierarchies and tag usage before introducing styles.

## Practice Ideas
- Build a multi-page portal using relative hyperlinks to travel between local files.
