# Links & Images Notes

## Important Concepts & Tags
- `<a>`: Defines a hyperlink to another resource.
- `href`: Anchor attribute specifying the target URL or file path.
- `<img>`: Embeds an image (self-closing tag).
- `src`: Image attribute specifying the path to the image resource.
- `alt`: Image description for accessibility and fallback.

## Common Mistakes
- Forgetting the `alt` attribute on `<img>` tags (bad for SEO and screen-readers).
- Mixing up relative paths (e.g. referencing a folder up or down incorrectly).
- Leaving `href` empty or omitting `http://` or `https://` for external links.

## Interview Notes
- **What is the difference between relative and absolute paths?** Absolute paths use the full URL (including protocol), whereas relative paths point to files relative to the current file's location.
- **What is the target="_blank" attribute?** It tells the browser to open the linked document in a new window or tab.
