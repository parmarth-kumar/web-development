Challenge: Newspaper Layout Outline

Apply the concepts you have learned to complete this challenge independently:

  • Structure a newspaper layout using only structural containers.
  • Use distinct `div` containers to divide the page into a Heading Block, Sidebar, and Main Article Block.
  • Within the article text, highlight specific keywords using `span` inline elements with inline styles (e.g., custom text color).
Remember: Avoid copy-pasting code or using step-by-step tutorials. The goal is to figure it out from your own knowledge!
