# HTML Boilerplate Notes

## Important Concepts & Tags
- `<!DOCTYPE html>`: Declares the document type as HTML5.
- `<html>`: The root container element.
- `<head>`: Contains document metadata, character set, viewport, and links to stylesheets/scripts.
- `<title>`: Defines the page title shown in browser tabs.
- `<body>`: Contains all visible page contents (headings, paragraphs, images, etc.).
- `<meta>`: Defines character encoding (`UTF-8`) and responsive layout configuration.

## Common Mistakes
- Forgetting the `<!DOCTYPE html>` declaration.
- Placing visible content tags like `<h1>` or `<p>` inside the `<head>` tag.
- Typing structural elements out of order.

## Interview Notes
- **What is the purpose of the DOCTYPE declaration?** It informs the web browser about the type and version of HTML being used, ensuring the browser renders the page in standards mode instead of quirks mode.
- **Why is `<meta name="viewport">` important?** It tells the browser how to control the page's dimensions and scaling on different screen sizes (specifically mobile viewport resets).
