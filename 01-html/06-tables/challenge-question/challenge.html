Challenge: Weekly Timetable Layout

Apply the concepts you have learned to complete this challenge independently:

  • Build a weekly class timetable table.
  • The header row should contain the days of the week.
  • Create time slots as rows.
  • Use `colspan` to merge columns for classes that stretch across multiple slots.
  • Use `rowspan` to span lectures spanning multiple hours.
Remember: Avoid copy-pasting code or using step-by-step tutorials. The goal is to figure it out from your own knowledge!
