# CSS Grid Notes

## Important Concepts & Tags
- `display: grid`: Activates grid container context.
- `grid-template-columns`: Specifies column configurations (e.g. using `1fr` or `repeat()`).
- `grid-gap` / `gap`: Sets cell intervals.
- `grid-area`: Names grid elements for layout mapping.

## Common Mistakes
- Mixing up row and column layouts inside grid templates.
- Overcomplicating layouts with flexbox when standard Grid rows and columns fit better.

## Interview Notes
- **When should you use Flexbox vs. CSS Grid?** Use Flexbox for one-dimensional layouts (either a row OR a column). Use CSS Grid for two-dimensional layouts (both rows AND columns simultaneously).

## Practice Ideas
- Design a structured photo grid featuring cells of unequal widths.
