Challenge: Reset Form Typography via Inheritance

Apply the concepts you have learned to complete this challenge independently:

  • Create an HTML layout with a styled parent div containing text and forms.
  • Apply a custom font and color to the parent div.
  • Apply styles to the inputs and buttons inside using `font-family: inherit` and `color: inherit` to demonstrate force-inheriting parent typography.
Remember: Avoid copy-pasting code or using step-by-step tutorials. The goal is to figure it out from your own knowledge!
