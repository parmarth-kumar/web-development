Challenge: Dynamic Grid Cards

Apply the concepts you have learned to complete this challenge independently:

  • Create a layout featuring three card components.
  • Create class selectors to style card backgrounds and text colors.
  • Use pseudo-classes (`:hover`) to make card backgrounds shift colors and add custom shadows when hovered over.
Remember: Avoid copy-pasting code or using step-by-step tutorials. The goal is to figure it out from your own knowledge!
